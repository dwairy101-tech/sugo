# نسخة SUGO مقسّمة وخفيفة

## شو تعمل على GitHub؟
ارفع كل محتويات المجلد كما هي، وليس ملف `index.html` وحده:

```
index.html
assets/
  css/main.css
  js/*.js
worker.js
```

مهم جداً: لا تغيّر اسم مجلد `assets` ولا أماكن الملفات داخله، لأن `index.html` مربوط عليها بهذه المسارات.

## ماذا تم عمله؟
- فصلت ملفات CSS في: `assets/css/main.css`.
- فصلت JavaScript إلى ملفات صغيرة مرتبة داخل: `assets/js/`.
- أبقيت كود شاشة البداية الضروري داخل `index.html` حتى تظهر مباشرة بدون فلاش.
- أضفت `defer` لملفات JavaScript حتى لا توقف قراءة HTML في البداية.
- نسخت `worker.js` كما هو بدون تغيير.

## النتيجة
- صار `index.html` تقريباً 87KB بدل ملف كبير جداً.
- صار المتصفح يقدر يعمل Cache لملفات CSS و JS، يعني الزيارة الثانية أخف.
- الرفع على GitHub لازم يكون كمجلد كامل بنفس الترتيب.
