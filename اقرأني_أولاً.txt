SUGO — النسخة المعدلة الجاهزة
=================================

الإصدار: 3.1.0-mode-aware-accuracy
التاريخ: 2026-07-13

ما تم إصلاحه:
- فصل نتائج Ask AI عن قوالب Create Ticket المخفية.
- تحسين فهم العبارات العربية العامية مثل: بدي أفتح وكالة، ما بقدر أفوت حسابي، انحظر حسابي، وما وصلني الشحن.
- منع قالب تذكرة ضعيف من تجاوز المقال الصحيح.
- تصحيح تقييم الثقة والحالات الغامضة.
- تحديث تعليمات الـWorker بحسب وضع Ask AI أو Create Ticket.
- إضافة اختبارات دقة جديدة وتشغيل جميع الاختبارات بنجاح.

طريقة الرفع على GitHub:
1. فك ضغط الملف.
2. ارفع جميع الملفات والمجلدات الموجودة داخله إلى جذر مستودع GitHub واستبدل الملفات القديمة.
3. انتظر اكتمال GitHub Pages ثم افتح الموقع واضغط Ctrl+F5.

ملاحظة Cloudflare:
- الإصلاح الرئيسي لمحرك البحث موجود في ملفات الواجهة وسيعمل بعد رفع النسخة على GitHub.
- مجلد worker يحتوي نسخة Worker محدثة، لكن لا تنفذ النشر إلا بعد تسجيل Wrangler بالحساب الصحيح الذي يحتوي على Worker باسم sugo.
- لا يوجد Account ID كامل داخل الحزمة لأن الحساب الصحيح يجب اختياره من لوحة Cloudflare الخاصة بك.
