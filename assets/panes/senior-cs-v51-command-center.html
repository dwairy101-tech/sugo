Use this page as the operating hub for Senior CS — Small & Sub Agency cases. Start with the search box, review the Best Match card, then open the relevant SOP before replying.

  • Check Best Match and confidence.

Internal Note

Decision flow:

  1. Identify the case type: activation, joining, commission, tasks, rating, activity rewards, or backend access.
  2. Open the matching Senior CS SOP.
  3. Confirm required identifiers before escalation: Agency ID, AGM ID, Anchor ID if applicable, screenshots, date range, and issue description.
  4. Use SOP Only mode for sensitive policy questions.
  5. Escalate only after required checks are complete.

ملاحظة داخلية

استخدم هذه الصفحة كمركز تشغيل لحالات Senior CS — Small & Sub Agency. ابدأ من مربع البحث، راجع بطاقة Best Match، ثم افتح إجراء SOP المناسب قبل الرد.

آلية العمل المقترحة:

  • ابحث باستخدام كلمات مشكلة المستخدم.
  • راجع أفضل تطابق ومستوى الثقة.
  • افتح مصدر SOP المناسب.
  • انسخ التذكرة المخصصة للعميل فقط عند الرد.
  • استخدم ملاحظات CS والتصعيد داخلياً فقط.
المنشن / التصعيد

تدفق القرار:

  1. حدد نوع الحالة: تفعيل، انضمام، عمولة، مهام، تصنيف، مكافآت أنشطة، أو دخول للنظام الخلفي.
  2. افتح إجراء Senior CS المطابق.
  3. تأكد من توفر البيانات المطلوبة قبل التصعيد: رقم الوكالة، رقم AGM، رقم المذيعة إن وجد، لقطات شاشة، الفترة الزمنية، ووصف المشكلة.
  4. استخدم وضع SOP Only للأسئلة الحساسة المتعلقة بالسياسات.
  5. لا يتم التصعيد إلا بعد اكتمال الفحوصات المطلوبة.
