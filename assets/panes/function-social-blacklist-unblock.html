To see the list of users you have blocked (blacklist) and to unblock someone, follow these steps:

  • Go to "Settings" (ME → Settings).
  • Tap on "Privacy Settings".
  • Select "Blacklist" (or "Blocked Users").
  • You will see a list of all users you have blocked.
  • To unblock, tap on the user's name or avatar and select "Unblock".

⚠️ Daily unblock limit: To prevent misuse, there is a daily limit on how many users you can unblock. If you exceed this limit, you will see a notification and must wait until the next day.

  • Unblocking allows that user to interact with you again. They will not be notified of the unblock.
  • You can re-block a user at any time after unblocking.

Regularly review your blacklist to keep it clean.

  • Managing your blacklist is simple. Here’s how to find it and remove a user:
  • Select "Blacklist" to see all the users you have blocked.
  • Find the user you wish to unblock and select the option to remove them from the list.
  • To prevent misuse of this feature, there is a daily limit on how many users you can remove from your blacklist.

  • انتقل إلى "الإعدادات" (أنا ← الإعدادات).
  • اضغط على "إعدادات الخصوصية".
  • اختر "القائمة السوداء" (أو "المستخدمون المحظورون").
  • سترى قائمة بجميع المستخدمين الذين قمت بحظرهم.
  • لإلغاء الحظر، اضغط على اسم المستخدم أو صورته الرمزية واختر "إلغاء الحظر".

⚠️ الحد اليومي لإلغاء الحظر: لمنع سوء الاستخدام، هناك حد يومي لعدد المستخدمين الذين يمكنك إلغاء حظرهم. إذا تجاوزت هذا الحد، فسترى إشعارًا ويجب عليك الانتظار حتى اليوم التالي.

  • إلغاء الحظر يسمح لذلك المستخدم بالتفاعل معك مرة أخرى. لن يتم إخطارهم بإلغاء الحظر.
  • يمكنك إعادة حظر مستخدم في أي وقت بعد إلغاء الحظر.

راجع قائمتك السوداء بانتظام للحفاظ عليها نظيفة.
