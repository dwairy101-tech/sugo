English

Game access requests are submitted through the Games Request Sheet. Removal requests do not have eligibility requirements.

Android game access requirements:

  • Host: Charm Level 3.
  • Supporter: VIP 4.
  • Spending more than 4,000 coins outside games, such as in Sound Rooms, during last week or this week gives access for the current week.
  • Spending more than 40,000 coins outside games gives permanent game access.

iPhone game access requirements:

  • Host: Charm Level 2.
  • Supporter: VIP 3.
  • CS collects the request and submits it through this sheet: Games Request Sheet
  • For Android users: If you are a host, you must have an Charm Level 3, or if you are a supporter, your account must be VIP 4.
  • If a user spends more than 4,000 coins (in other areas such as the Sound Room, etc.) last week or this week, they will have access to the games this week.
  • If a user spends more than 40,000 coins (in other areas such as the Sound Room, etc.), they will have permanent access to the games.
  • For iPhone users: If you are a host, your Charm Level 2 is required, and if you are a supporter, your account must be VIP 3.
  • Remove games: no requirement, submit the request through the sheet.
العربية

طلبات الوصول إلى الألعاب تُرسل من خلال Games Request Sheet. طلبات إزالة الألعاب لا تحتاج إلى شروط أهلية.

شروط الوصول للألعاب على Android:

  • المضيف: مستوى Charm 3.
  • الداعم: VIP 4.
  • إنفاق أكثر من 4,000 كوين خارج الألعاب، مثل غرف الصوت، خلال الأسبوع الماضي أو هذا الأسبوع يمنح وصولًا للألعاب خلال الأسبوع الحالي.
  • إنفاق أكثر من 40,000 كوين خارج الألعاب يمنح وصولًا دائمًا للألعاب.

شروط الوصول للألعاب على iPhone:

  • المضيف: مستوى Charm 2.
  • الداعم: VIP 3.
