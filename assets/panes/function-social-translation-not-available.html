Welcome to the SUGO family!

We sincerely apologize for the issue you are experiencing.

If you are unable to translate a message or receive a notification that the daily translation limit has been reached, here is why:

  • The translation feature has a daily limit of 55 uses per user. This limit helps manage server load and prevent abuse.
  • Once you exceed 55 translations in a single day, the feature becomes unavailable until the next day.
  • The limit resets at midnight (00:00) based on your local time zone (or UTC).
  • Translation is available for most languages, but some less common languages may not be supported.

To use translation efficiently, only translate messages you truly do not understand.

If you need to translate more than 55 messages per day, consider using an external translation app as an alternative.

Thank you for understanding this reasonable limit.

Best regards,

Sugo Support Team

  • If you are unable to translate other users’ messages or receive a notification saying that the daily translation limit has been reached, it is because there is a usage limit for the translation feature.
  • Each user can use the translation function up to 55
  • Once this limit is exceeded, the translation feature will be temporarily unavailable
  • You will see a reminder indicating that the daily limit has been reached
  • We recommend trying again the next day, when your translation quota has been reset.

  • تحتوي ميزة الترجمة على حد يومي قدره 55 استخدامًا لكل مستخدم. يساعد هذا الحد في إدارة حمل الخادم ومنع إساءة الاستخدام.
