Apply to Open Host Agency

Answer

To create a host agency inside the app, please make sure all the following requirements are fully met:

  1. The minimum requirement to open an agency is five new hosts.
  2. The host accounts must be new and used on new devices.
  3. The applicant must not currently be a host.
  4. The applicant must not currently be a sub-agency agent.
  5. Each host account must have been registered less than 14 days ago.

If all requirements are met, please send the following information in full:

  • Your account ID.
  • The IDs of the five hosts.
  • Your real full name.
  • An active WhatsApp number with the international country code.
  • Country.

Please note that if you are currently a host or a sub-agency agent, the request cannot be submitted. The relevant management team will contact you within 24 to 48 hours via WhatsApp to explain all details.
