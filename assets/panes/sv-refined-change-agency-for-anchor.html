Change Agency for Anchor

Answer

If you are requesting to transfer from one agency to another, please follow the conditions below:

  1. If the current agent approves:

The transfer will be completed directly by the current agent according to the approved procedures.

  1. If the current agent does not approve:

The monthly target must be stopped for two full months. After this period ends, a transfer request can be submitted to the new agent.

  1. If there is abuse from the agent:

Please record a clear video showing the abuse, the agent's ID, and your own account ID.

  1. If the agent does not respond:

If the agent does not respond on WhatsApp and inside the app for at least three consecutive days, please record both conversations in full, showing the agent's ID and your account ID, then send the full video to the new agent so the required action can be taken.

Please note that if the WhatsApp conversation is not available, the transfer request cannot be submitted. All requirements must be followed before the request can be reviewed.
