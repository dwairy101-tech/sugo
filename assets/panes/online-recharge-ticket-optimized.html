Online Recharge Optimized

Use this page for online recharge problems, payment proof, missing coins, and recharge follow-up.

Required information before checking

  • User ID.
  • Clear description of the issue.
  • Screenshot or screen recording.
  • Date and time of the issue.
  • Related IDs when available: room ID, order ID, agency ID, host ID, or ticket ID.
  • Exact error message if shown.

CS decision flow

  1. Identify the exact ticket type.
  2. Open the matching SOP macro in this section.
  3. Collect all required evidence before escalation.
  4. Use customer-facing Ticket text for the user.
  5. Keep internal notes, mentions, and escalation instructions for CS use only.
  6. If the case is unclear, ask for missing details instead of guessing.

Escalation

  • Escalate only after the correct ID, screenshots, and timeline are collected.
  • For policy or violation cases, use SOP Only mode and avoid unsupported explanations.
  • For technical cases, include device model, app version, and screen recording.

Customer reply template

Welcome to the SUGO family!

We sincerely apologize for the issue you are experiencing.

To help us check this case accurately, please provide your user ID, a clear description of the issue, a screenshot or screen recording, and the date/time when the issue occurred.

If there is any related ID such as room ID, order ID, agency ID, or host ID, please include it as well.

Thank you.

المعلومات المطلوبة قبل الفحص

  • رقم المستخدم ID.
  • وصف واضح للمشكلة.
  • لقطة شاشة أو تسجيل شاشة.
  • تاريخ ووقت حدوث المشكلة.
  • الأرقام المرتبطة عند توفرها: رقم الغرفة، رقم الطلب، رقم الوكالة، رقم المذيعة، أو رقم التذكرة.
  • رسالة الخطأ الدقيقة إن وجدت.

خطوات قرار خدمة العملاء

  1. تحديد نوع التذكرة بدقة.
  2. فتح SOP أو الماكرو المطابق داخل هذا القسم.
  3. جمع جميع الأدلة المطلوبة قبل التصعيد.
  4. استخدام نص Ticket الموجه للعميل فقط عند الرد.
  5. إبقاء الملاحظات الداخلية والمنشن وتعليمات التصعيد للاستخدام الداخلي فقط.
  6. إذا كانت الحالة غير واضحة، يتم طلب المعلومات الناقصة بدلاً من التخمين.

التصعيد

  • يتم التصعيد فقط بعد جمع رقم الحساب الصحيح ولقطات الشاشة والتوقيت.
  • في حالات السياسات أو المخالفات، استخدم وضع SOP Only وتجنب أي تفسير غير مدعوم.
  • في الحالات التقنية، أرفق نوع الجهاز وإصدار التطبيق وتسجيل الشاشة.
