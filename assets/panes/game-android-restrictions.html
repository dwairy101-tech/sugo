Android Game Restrictions

Temporary Game Restriction

All games will be temporarily hidden if:

  • The user spends 2,000+ coins during the current week, or
  • The user spends 2,000+ coins during the previous week.

Permanent Game Restriction

All games will be permanently hidden if:

  • The user's total non-game consumption reaches 40,000+ coins.

قيود الألعاب على Android

تقييد مؤقت للألعاب

سيتم إخفاء جميع الألعاب مؤقتًا إذا:

  • أنفق المستخدم 2000+ عملة خلال الأسبوع الحالي، أو
  • أنفق المستخدم 2000+ عملة خلال الأسبوع الماضي.

تقييد دائم للألعاب

سيتم إخفاء جميع الألعاب بشكل دائم إذا:

  • إجمالي استهلاك المستخدم خارج الألعاب يصل إلى 40000+ عملة.
