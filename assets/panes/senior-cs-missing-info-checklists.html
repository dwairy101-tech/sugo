Missing Info Checklists

Answer

Use these checklists before escalation or before giving a final answer.

Agency verification delay:

  • Agency ID.
  • AGM ID.
  • Verification submission date.
  • Screenshot of pending status.

Commission or revenue issue:

  • Agency ID.
  • Date range.
  • Screenshot of commission/revenue page.
  • Expected amount and actual amount.
  • Rating record if relevant.

Anchor joining/rejection issue:

  • Anchor ID.
  • Agency ID.
  • Application status screenshot.
  • Rejection reason if visible.
  • Whether face verification was completed.

Transfer code request:

  • Correct AGM ID.
  • Agency ID.
  • Screenshot of missing transfer code or request page.
  • Confirmation that the request is sent from the correct ID.

Activity reward issue:

  • Activity name.
  • Room ID.
  • Anchor ID.
  • Event time.
  • Proof video including room ID, anchor ID, and last minute of stream.

ملاحظة داخلية

استخدم هذه القوائم قبل التصعيد أو قبل إعطاء رد نهائي.

تأخر التحقق من الوكالة:

  • رقم الوكالة.
  • رقم AGM.
  • تاريخ إرسال طلب التحقق.
  • لقطة شاشة لحالة الطلب المعلق.

مشكلة عمولة أو إيراد:

  • رقم الوكالة.
  • الفترة الزمنية.
  • لقطة شاشة من صفحة العمولة/الإيراد.
  • المبلغ المتوقع والمبلغ الفعلي.
  • سجل التقييم إذا كان مرتبطاً بالحالة.

مشكلة انضمام أو رفض مذيعة:

  • رقم المذيعة.
  • رقم الوكالة.
  • لقطة شاشة لحالة الطلب.
  • سبب الرفض إذا كان ظاهراً.
  • تأكيد ما إذا تم إكمال التحقق من الوجه.

طلب رمز التحويل:

  • رقم AGM الصحيح.
  • رقم الوكالة.
  • لقطة شاشة لعدم وجود الرمز أو صفحة الطلب.
  • التأكد من أن الطلب مرسل من الرقم الصحيح.

مشكلة مكافأة نشاط:

  • اسم النشاط.
  • رقم الغرفة.
  • رقم المذيعة.
  • وقت الحدث.
  • فيديو إثبات يتضمن رقم الغرفة، رقم المذيعة، والدقيقة الأخيرة من البث.
