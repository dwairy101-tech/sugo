Account Support Optimized

Use this page as the first triage checklist for account-related cases.

When to use

  • Registration or login issues.
  • Password recovery.
  • Linked phone/email issues.
  • Account access or account security questions.
  • Cases where the user cannot access the original login method.

Required information before checking

  • User ID.
  • Registered phone/email/login method.
  • Screenshot or screen recording of the error.
  • Device model and app version when relevant.
  • Whether the user still has access to the linked phone/email.

CS decision flow

  1. Confirm the account ID and issue type.
  2. Identify the current login method.
  3. Check whether the user has access to the linked phone/email.
  4. If the issue is caused by lost access, follow the recovery or binding SOP.
  5. If ownership is unclear, request verification evidence before escalation.
  6. Never disclose sensitive account information to an unverified requester.

Escalation

  • Escalate only after the correct ID and evidence are collected.
  • For suspicious ownership or security cases, do not promise recovery before review.

Customer reply template

Welcome to the SUGO family!

We sincerely apologize for the issue you are experiencing.

To help us check your account issue accurately, please provide your user ID, the login method you used, a screenshot of the error, and let us know whether you still have access to the linked phone/email.

Once we receive the required information, we will review the issue according to the account support procedure.

Thank you.

Account Support Optimized

استخدم هذه الصفحة كقائمة فحص أولية لحالات الحساب.

متى يُستخدم هذا القسم

  • مشاكل التسجيل أو تسجيل الدخول.
  • استعادة كلمة المرور.
  • مشاكل رقم الهاتف أو البريد المرتبط.
  • أسئلة الوصول إلى الحساب أو أمان الحساب.
  • الحالات التي لا يستطيع فيها المستخدم الوصول إلى طريقة الدخول الأصلية.

المعلومات المطلوبة قبل الفحص

  • رقم المستخدم ID.
  • رقم الهاتف/البريد/طريقة تسجيل الدخول.
  • لقطة شاشة أو تسجيل شاشة للخطأ.
  • نوع الجهاز وإصدار التطبيق عند الحاجة.
  • توضيح هل ما زال المستخدم يملك الوصول إلى الهاتف أو البريد المرتبط.

خطوات قرار خدمة العملاء

  1. تأكيد رقم الحساب ونوع المشكلة.
  2. تحديد طريقة تسجيل الدخول الحالية.
  3. التحقق مما إذا كان المستخدم يملك الوصول إلى الهاتف أو البريد المرتبط.
  4. إذا كانت المشكلة بسبب فقدان الوصول، يتم اتباع إجراء الاسترداد أو الربط المناسب.
  5. إذا لم تكن ملكية الحساب واضحة، يتم طلب أدلة تحقق قبل التصعيد.
  6. لا يتم الإفصاح عن معلومات حساسة لأي شخص غير موثّق.

التصعيد

  • يتم التصعيد فقط بعد جمع رقم الحساب الصحيح والأدلة المطلوبة.
  • في حالات الشك بملكية الحساب أو الأمان، لا يتم الوعد باسترداد الحساب قبل المراجعة.

قالب رد للعميل

مرحباً،

حتى نتمكن من فحص مشكلة حسابك بدقة، يرجى تزويدنا برقم المستخدم، وطريقة تسجيل الدخول المستخدمة، ولقطة شاشة للخطأ، وتوضيح ما إذا كنت لا تزال تملك الوصول إلى رقم الهاتف أو البريد المرتبط.

بعد استلام المعلومات المطلوبة، سنقوم بمراجعة الحالة وفق إجراءات دعم الحساب.
