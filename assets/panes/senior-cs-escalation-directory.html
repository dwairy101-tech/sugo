Mention / Escalation

Escalation map:

  • Delayed agency verification: Senior Monitor Team.
  • Voluntary agency closure / transfer active hosts: Close Agency and Transfer Anchors Form + @Charlotte.
  • Missing AGM transfer code: ACM group + @Charlotte.
  • AGM dashboard password reset: Audition Team.
  • BCM anchor transfer request: HQ.
  • Activity reward missing after weekly payout: Management/activity owner with proof video.
  • Commission mismatch: collect Agency ID, screenshots, date range, expected amount, and commission/rating records before escalation.
Internal Form

Before escalating, collect:

  • Case type.
  • Agency ID.
  • AGM ID / user ID.
  • Anchor ID if host-related.
  • Screenshot or screen recording.
  • Date range and expected value if revenue/commission is involved.
  • Short summary of what was already checked.

المنشن / التصعيد

خريطة التصعيد:

  • تأخر التحقق من الوكالة: فريق Senior Monitor.
  • إغلاق الوكالة طوعياً / نقل المذيعات النشيطات: نموذج Close Agency and Transfer Anchors Form + منشن @Charlotte.
  • عدم توفر رمز تحويل AGM: مجموعة ACM + منشن @Charlotte.
  • إعادة تعيين كلمة مرور لوحة AGM: فريق Audition.
  • طلب نقل مذيعة BCM: HQ.
  • عدم استلام مكافأة نشاط بعد موعد الصرف الأسبوعي: الإدارة/مسؤول النشاط مع فيديو الإثبات.
  • اختلاف العمولة: جمع رقم الوكالة، لقطات الشاشة، الفترة الزمنية، المبلغ المتوقع، وسجلات العمولة/التقييم قبل التصعيد.
ملاحظة داخلية

قبل التصعيد، يجب جمع:

  • نوع الحالة.
  • رقم الوكالة.
  • رقم AGM / رقم المستخدم.
  • رقم المذيعة إذا كانت الحالة مرتبطة بمذيعة.
  • لقطة شاشة أو تسجيل شاشة.
  • الفترة الزمنية والقيمة المتوقعة إذا كانت الحالة متعلقة بالإيراد أو العمولة.
  • ملخص قصير لما تم فحصه مسبقاً.
