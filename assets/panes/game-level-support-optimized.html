Game Level Requirement Optimized

Use this page when users ask why a game feature is locked, unavailable, or requires a certain level.

Required information before checking

  • User ID.
  • Game name.
  • Current level.
  • Screenshot of the lock/error/requirement.
  • Device model and app version if the issue looks technical.

CS decision flow

  1. Confirm the game or feature name.
  2. Check the required level or eligibility shown in the app.
  3. Compare it with the user’s current level.
  4. If the user meets the requirement but the feature remains locked, collect screenshots and escalate.
  5. If the user does not meet the requirement, explain the requirement clearly.

Customer reply template

Welcome to the SUGO family!

We sincerely apologize for the issue you are experiencing.

Please provide your user ID, the game name, your current level, and a screenshot of the requirement or error message.

If you already meet the requirement and the game is still unavailable, we will check it further.

Thank you.

المعلومات المطلوبة قبل الفحص

  • رقم المستخدم.
  • اسم اللعبة.
  • المستوى الحالي.
  • لقطة شاشة من القفل أو الخطأ أو المتطلبات.
  • نوع الجهاز وإصدار التطبيق إذا بدت المشكلة تقنية.

خطوات قرار خدمة العملاء

  1. تأكيد اسم اللعبة أو الميزة.
  2. مراجعة المستوى أو شرط الأهلية الظاهر داخل التطبيق.
  3. مقارنة الشرط بالمستوى الحالي للمستخدم.
  4. إذا كان المستخدم مستوفياً للشرط وما زالت الميزة مقفلة، يتم جمع لقطات الشاشة والتصعيد.
  5. إذا لم يكن مستوفياً للشرط، يتم توضيح المتطلبات بوضوح.
