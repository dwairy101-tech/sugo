Remove Withdrawal / Exchange Option

Process

  1. Confirm whether the user has any pending withdrawal transactions.
  2. Inform the user that disabling the option will cause any pending withdrawal request to fail.
  3. The option can only be removed after the withdrawal transaction fails for this reason.
  4. Submit the request through:

https://micoworld.feishu.cn/share/base/form/shrcnm0SCrveAVtp0l5m25MBF8e?from=from_parent_docs

العربية

إزالة خيار السحب / الاستبدال

الإجراء

  1. تأكد مما إذا كان المستخدم لديه معاملات سحب معلقة.
  2. أعلم المستخدم بأن تعطيل الخيار سيؤدي إلى فشل أي طلب سحب معلق.
  3. لا يمكن إزالة الخيار إلا بعد فشل معاملة السحب لهذا السبب.
  4. قدّم الطلب عبر:

https://micoworld.feishu.cn/share/base/form/shrcnm0SCrveAVtp0l5m25MBF8e?from=from_parent_docs
