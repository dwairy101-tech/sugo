# High-resolution Visual Guides

- All 142 built-in instructional screenshots were increased to 4× their original pixel dimensions.
- File names and paths were preserved, so no JavaScript or mapping changes are required.
- Processing used lossless PNG output, Lanczos resampling, and mild non-generative sharpening.
- The Worker, Cloudflare secrets, KV binding, and administrator media manager were not changed.
- Newly uploaded administrator images are stored at their original uploaded resolution.

Important: increasing pixel dimensions improves display and zoom clarity, but details absent from the original screenshot cannot be recreated exactly.
